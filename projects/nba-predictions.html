NBA Game Score Prediction

Personal project · 2021 — ongoing

Overview

A personal project forecasting the final scores of NBA games before tip-off. The model takes in pre-game information — team records, recent form, home/away splits, rest days, and head-to-head history — and outputs predicted scores for both teams. Predictions are logged publicly and archived by season for post-hoc accuracy analysis.

Approach

The prediction pipeline has two parallel outputs trained on historical game data going back to 2010:

  • Score regression — a gradient boosting regressor predicts the raw point totals for each team, fit on rolling per-game stats (offensive and defensive rating, pace, three-point rate, turnover rate) with engineered features for rest differential and home-court advantage
  • Win classification — a separate logistic regression model predicts the win probability for each team, used as a calibration check on the regression output

Features are updated nightly from the Basketball Reference API. Predictions are generated for every game on the schedule by noon ET on game day.

Performance

Score predictions are evaluated on mean absolute error (MAE) against the final box score. Win/loss direction is evaluated by whether the predicted winner matches the actual winner.

~8.4 Points MAE (2021–22)
~63% Win direction accuracy
2 Seasons archived
